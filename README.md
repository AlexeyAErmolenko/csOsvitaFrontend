ДЗ #1: Створення базової HTML-структури магазину кави<br>
Ваше завдання — створити HTML-розмітку для однієї картки товару, використовуючи вивчені теги.<br>
Зробіть просту версію з мінімальною кількістю елементів. Наразі не переймайтеся стилями та позиціонуванням — зосередьтеся на структурі та семантиці розмітки.<br>
Здавайте роботу через Git: надішліть посилання на ваш репозиторій у особисті повідомлення.<br>
